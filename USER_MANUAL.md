# 自動簽到模組 (SignIn) 可視化配置界面使用手冊

本模組提供完整的自動化配置界面，您可以通過以下方式進入 UI 配置介面：

## 1. 如何進入介面

您可以通過以下兩種方式打開配置介面：

- **方式一 (推薦)**：在遊戲中輸入指令 `/dailyrewards config`。
- **方式二**：在安裝了 `Mod Menu` 模組後，在遊戲主選單的「模組」列表內找到 `SignIn`，點擊「配置」按鈕。

## 2. 介面功能概覽

介面分為四大區塊，方便您快速定位配置項：

### A. 基礎參數 (General)

- **自動退出伺服器**：任務完成後是否自動中斷連線。
- **自動放置物品**：是否自動將快捷欄物品存入最近的箱子。
- **自動執行回家命令**：任務結束後是否自動執行 `/home` 等指令。
- **自動進入伺服器**：啟動任務時是否自動連接到設定的伺服器。
- **開箱物品欄位**：自動存物時玩家手持的快捷欄索引 (0-8)。

### B. 指令設置 (Commands)

- 可在此修改登入、簽到、回家及退出指令。支持實時錯誤檢測。

### C. 帳號與伺服器管理 (Accounts & Servers)

- **列表管理**：支持通過 UI 列表新增、刪除或排序多個帳號與伺服器地址。
- **索引設定**：直接拖動數字設置起始執行的帳號與伺服器。

## 3. 操作與安全提示

- **實時保存**：在 UI 中修改的任何選項都會立即自動保存至磁盤，無需手動點擊保存按鈕。
- **變更提示**：修改後螢幕右上角會彈出 Toast 提示訊息確認操作。

## 4. 審計與安全性

- **操作日誌**：所有通過 UI 或指令進行的修改都會記錄在 `.minecraft/config/signin_audit.log` 中，包含時間、操作人與內容。
- **敏感數據脫敏**：在日誌顯示中，系統會自動識別並隱藏指令中的密碼部分。

## 5. 常見問題

- **Q: 為什麼我的修改沒有生效？**
  - A: 由於系統採用實時保存機制，若修改後立即強制關閉遊戲可能導致緩存未寫入。通常修改後看到右上角提示即代表已生效。
- **Q: 如何快速切換帳號？**
  - A: 在「帳號管理」區塊中，點擊左右箭頭或直接輸入數字。
